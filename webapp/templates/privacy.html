Privacy Policy

Last updated: {{ updated }}

Socials Auto-Publisher ("the App") is a personal automation tool with a single operator, Mykhailo Kolomoitsev ("the Operator"). It is not a public product and does not collect data from, or about, any member of the public.

1. What data the App processes

  • Video files sent by the Operator via a private Telegram bot.
  • Generated artifacts: subtitle files, processed video, and cover images derived from the submitted video.
  • Transcript text produced by an automated speech-to-text service, used to generate subtitles and captions.
  • Publishing metadata: timestamps, scheduling information, and view counts retrieved from the TikTok API.

The App does not collect data from end viewers of the published videos, and does not process content from any user other than the Operator.

2. How data is stored

Processed video and cover files are stored in a private cloud storage bucket (Amazon S3 or a compatible provider) and referenced by URL when publishing to TikTok and Instagram. Publishing history and scheduling state are stored in a local database used only by the App.

3. Third-party services used

  • Telegram — for receiving video submissions from the Operator.
  • OpenAI Whisper or AssemblyAI — for automated speech-to-text transcription.
  • Amazon S3 (or compatible storage) — for temporary hosting of media files referenced during publishing.
  • TikTok Content Posting API — for publishing videos and retrieving view counts on the Operator's own TikTok account.
  • Instagram Graph API (Meta) — for publishing Reels to the Operator's own Instagram account.

Each of these services processes data under its own privacy policy. The App only sends them the minimum data required to perform the requested action (e.g., the video file for transcription or publishing).

4. Data retention

Media files are retained in cloud storage and in the App's local database for as long as needed to support the publishing workflow and view-count comparison. The Operator can delete this data at any time.

5. Data sharing

The App does not sell, rent, or share data with any party other than the third-party services listed above, which are required to perform the App's core function.

6. Security

Access to the App is restricted to a single authorized Telegram user ID. API credentials are stored as environment variables and are not committed to source control.

7. Contact

Questions about this Privacy Policy can be sent to mihon.testosterone@gmail.com.
